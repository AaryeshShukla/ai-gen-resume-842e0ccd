# Aaryesh Shukla - Interactive Resume

## Overview
This is a premium interactive resume for Aaryesh Shukla, designed to showcase skills, projects, and experience in a visually stunning manner using modern web technologies.

## Features
- Glassmorphic dark theme with smooth transitions
- Sticky navigation bar with active section indicators
- Interactive project showcase with modals for detailed views
- Skill proficiency indicators
- Contact form with real-time validation

## Technologies Used
- HTML, CSS, JavaScript
- Google Fonts (Outfit, Inter)

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser to view the resume.
